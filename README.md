# Ченнелинг-группа с Марией Шишкиной — лендинг

Одностраничный сайт ченнелинг-группы. Статичный HTML/CSS/JS, без сборки и зависимостей.
Свёрстан по макету из Figma «Внутривседома».

## Структура

```
index.html          — вся разметка страницы
css/styles.css      — стили и адаптив (десктоп + мобильная версия)
js/main.js          — логика формы регистрации
assets/img/         — фото и коллажные элементы из макета
.github/workflows/  — авто-деплой на GitHub Pages
```

## Локальный просмотр

Просто откройте `index.html` в браузере, либо поднимите простой сервер:

```bash
python3 -m http.server 8000
# затем откройте http://localhost:8000
```

## Регистрация (GetCourse)

Кнопка «Зарегистрироваться» и форма заявки ведут на GetCourse.
Откройте `js/main.js` и впишите ссылку в начале файла:

```js
const REGISTRATION_URL = "https://ваша-школа.getcourse.ru/...";
```

- Если ссылка указана — кнопка в первом экране и кнопка формы открывают GetCourse
  (имя, почта и телефон передаются в параметрах ссылки).
- Если поле пустое — форма просто показывает сообщение «Заявка отправлена».

## Что меняется перед запуском

- **Дата и стоимость участия** — блок «Участие и регистрация» (`#registration` в `index.html`).
- **Ссылка регистрации** — `REGISTRATION_URL` в `js/main.js`.
- **Отзывы** (`assets/img/review1–4.png`) — пример из макета, можно заменить или убрать.

## Деплой

Пуш в ветку `main` автоматически публикует сайт через GitHub Pages
(workflow `.github/workflows/deploy.yml`). В настройках репозитория
**Settings → Pages → Source** должно стоять **GitHub Actions**.
